Trade log tool 便携版使用说明

1. 先确认这台电脑已经安装 Python 3.10 或更高版本。
   安装时建议勾选 Add Python to PATH。

2. 双击：
   启动 Trade log tool.bat

3. 启动后浏览器会自动打开：
   http://127.0.0.1:8765

4. 交易数据默认保存在：
   data\\trades.json

5. 导出 JSON 时，浏览器建议文件名会是：
   YYYYMMDD-HHMMSS-trades-export.json

6. 建议把导出的 JSON 手动保存到：
   exports_json

7. 如果要把旧数据带到另一台电脑：
   直接一起拷贝 data 文件夹，或者导入你之前导出的 JSON。

8. 关闭工具：
   关闭浏览器后，再回到启动窗口按 Ctrl + C。
